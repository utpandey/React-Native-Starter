## Description

This a Simple React Native App with basic implementation of Native functionalities like SafeAreaView,ScrollView, FlatList, React-Navigation (Stack Nav).

This contains all the stuff that I learned from FrontEnd Masters Workshop.


## :wrench: Installation


1. Clone / Download [this](https://github.com/utpandey/React-Native-Starter) repo.
2. Inside the project open a terminal and run:
    ```
    npm install
    ```
    This will install all the project dependencies.
3. On one terminal run :
    ```
    npm start
    ```
4. On another terminal run:
    ```
    npx react-native run-android
    ```
